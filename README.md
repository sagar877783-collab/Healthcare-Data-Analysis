# Healthcare Data Analysis

## Project Overview
This project analyzes healthcare patient data using Python.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Features
- Data Analysis
- Statistical Summary
- Data Visualization
- Healthcare Trend Analysis

## Files
- main.py
- healthcare_dataset.csv
- requirements.txt

## How to Run

Install libraries:

pip install -r requirements.txt

Run project:

python main.py
